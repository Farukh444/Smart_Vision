👁️ Smart Vision: Enhancing Independence with Object Recognition for the Visually Impaired

Smart Vision is an AI-based assistive technology that helps visually impaired individuals navigate their surroundings through real-time object and obstacle detection. The system uses a camera, computer vision, and deep learning models (CNN & YOLO) to identify objects and provide audio feedback, enabling safer and more independent mobility.

🚀 Features
Real-time Object Detection
Obstacle Recognition & Alerts
Audio Feedback using Text-to-Speech
CNN & YOLO-based Deep Learning Models
Wearable and Portable Design
Indoor & Outdoor Navigation Support
Low-cost and User-Friendly Solution
🛠️ Tech Stack
Programming Language: Python
Computer Vision: OpenCV
Deep Learning: YOLO, CNN, TensorFlow/PyTorch
Audio Processing: GTTS / pyttsx3
Hardware: Camera, Raspberry Pi / Jetson Nano, Headphones
Libraries: NumPy, Pandas, Matplotlib
🎯 Objective

To improve the independence, mobility, and safety of visually impaired individuals by providing real-time environmental awareness through intelligent object recognition and audio guidance.
